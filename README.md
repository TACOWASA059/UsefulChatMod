# UsefulChatMod
チャット便利系クライアントmod forge1.16.5

## 機能
- チャットのタイムスタンプの表示
- チャットをクリックで入力欄にテキストを追加
- チャットをshift+クリックでクリップボードにコピー
- チャット、/tellそれぞれで表示色を指定
- tellに通知音を追加
- tellのみが表示されるチャット欄の追加
- F9でコンフィグを開く
